# 🎵 AI Song Recommendation Web App

This is a web application that recommends songs using Artificial Intelligence.
The app uses the Google Gemini API to generate song suggestions based on user input such as mood, genre, or favorite artist.

## 🚀 Features

- AI-based song recommendations

- Uses Gemini API

- Simple and user-friendly interface

- Instant music suggestions

## 🛠️ Technologies Used

- HTML

- CSS

- Python

- Gemini API
  
- FastAPI

## 💡 How It Works

- User enters mood, genre, or artist

- The request is sent to Gemini API

- AI generates song recommendations

- Songs are displayed on the webpage
